# nodejs-web-vite-yarn-preact-typescript-template

## Description
A preact template for the vite packager.

## Tech stack
- yarn
- vite
- preact
- typescript

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
